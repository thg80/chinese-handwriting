# 后续更新计划

## 当前已实现（V1.2 复习字库）

- 随机抽字（拼音 + 例词提示）
- Canvas 手写板（鼠标/触摸）
- 自评熟悉度（会写 / 不会写）
- 生疏字优先复现（70% 概率抽取错题本）
- 本地进度持久化 + 学习日志（localStorage）
- 字库：11441 常用汉字，按《通用规范汉字表》frequency 0-5 分级
- 难度选择器（全部/常用/进阶/高级）
- 复习面板：按日期、状态筛选已学字，排序查看，点击跳转练习

## 更新原则

1. 优先做**低难度高体感**的功能，保持项目轻量。
2. 任何改动必须同步更新 `CHANGELOG.md`。
3. 涉及新文件夹或文件，遵循已有注释与架构说明规范。
4. 高复杂度功能在动手前应先确认是否有更轻量的替代方案。
