# js 文件夹

一旦我所属的文件夹有所变化，请更新我。

## 架构说明

存放业务逻辑与交互封装脚本。

## 文件清单

- `canvas.js` — 手写板驱动，负责 Canvas 初始化、笔迹绘制、触摸/鼠标事件、田字格离屏缓存加速、resize 防抖。（纯 ES5 语法，兼容旧版平板浏览器；提取了画布线宽、网格颜色等命名常量）
- `app.js` — 应用主逻辑，加载字库、难度过滤、基于熟悉度的加权抽字算法（含间隔复习待复习池）、答案揭示、熟悉度评价（会写/长按已认识/不会写）、学习日志记录、视图切换、批量复习队列、复习/练习模式切换、统计面板、数据导出/导入、键盘快捷键。（setStorage 带 try-catch，兼容 file:// 与隐私模式；提取了长按阈值、日志上限等命名常量）
- `review.js` — 复习面板，读取学习日志与熟悉度记录，提供日期/状态筛选（基于熟悉度阈值）、熟悉度排序、字符卡片网格（含熟悉度进度条），支持点击跳转练习。列表超过 200 张时分页加载，降低 DOM 节点数。
